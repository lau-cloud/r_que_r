2. Acceder a la API

Vamos a llamar a la API de Aemet con el paquete {climaemet} para acceder a los datos. Lo desarrollaron Manuel Pizarro, Diego Hernangómez y Gema Fernández-Avilés.


Primer paso: Instalar y cargar el paquete {climaemet}

library(tidyverse)
library(climaemet)
library(lubridate)
#install.packages("climaemet")

# si te aparece un aviso de que no tienes un paquete instalado, recuerda 
# que lo puedes instalar con install.packages("nombre_paquete")

Segundo paso: generar una clave para acceder a la API.

La siguiente línia de código nos abrirá una página en el navegador. Nos pedirán una dirección de correo, y la clave se nos enviará a esa dirección. (Copia y pega esta línea en la consola de R, y ejecútala)

browseURL("https://opendata.aemet.es/centrodedescargas/obtencionAPIKey")

Una vez tengamos la clave, la podemos registrar en R para que no tengamos que hacerlo cada vez que la usemos. (Copia la siguiente línea de código en la consola, y reemplaza lo que hay entre comillas por la clave que te ha llegado al correo. Luego ejecuta)

aemet_api_key("CLAVE_API_QUE_NOS_LLEGA_AL_CORREO", install = TRUE)

Una vez instalado, no tendremos que volver a hacer esta operación, porque se guardará la clave en nuestro PC. Ahora ya podemos ver los municipios disponibles y su código del INE:

3. Las previsiones

Para un municipio en concreto hay la previsión de hoy y los próximos seis días de las siguientes variables:

  • Temperatura

  • Sensación térmica

  • Probabilidad de precipitación

  • Cota de nieve por provincia

  • Estado del cielo

  • Viento

  • Racha máxima de viento

  • Humedad relativa

    Las mismas que encontraríamos en la interfaz de Aemet

Vamos a ver cómo han evolucionado las mínimas y máximas mensuales a nivel nacional

historico$mes <- format(as.Date(historico$fecha, format="%Y-%m-%d"),"%m")


datos_maximas <- historico |> 
  select(indicativo, tmax, tmin, year, mes)



## calcular media de mínimas y de máximas por década y mes
datos_maximas$year <- as.numeric(datos_maximas$year)
datos_maximas$mes <- as.numeric(datos_maximas$mes)


datos_clean <- datos_maximas |> 
  drop_na() |> 
  mutate(decada = case_when(year < 1990 ~ 1980,
                            year >= 1990 & year <2000 ~ 1990,
                            year >= 2000 & year < 2010 ~ 2000,
                            year >= 2010 & year < 2020 ~ 2010,
                            year >= 2020 ~ 2020)) |> 
  pivot_longer(cols = tmax:tmin,
               names_to = "variable",
               values_to = "valor") |> 
  group_by(decada, mes, variable) |>
  summarize(media = mean(valor))



##### plot
p <- ggplot(datos_clean, aes (x = decada, y = media, color = variable)) +
